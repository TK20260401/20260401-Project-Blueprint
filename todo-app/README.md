# 📝 シンプルTODOアプリ

シンプルで使いやすいWebベースのTODOアプリケーションです。

## ✨ 機能

- ✅ **タスク追加** - 新しいタスクを簡単に追加できます
- ✓ **タスク完了/未完了の切り替え** - チェックボックスでタスクの状態を管理
- 🗑️ **タスク削除** - 不要なタスクを削除
- 📊 **統計情報** - 全体のタスク数と完了数を表示
- 💾 **自動保存** - ローカルストレージにデータを自動保存（ブラウザを閉じても保持）
- 📱 **レスポンシブデザイン** - スマートフォンやタブレットにも対応

## 🚀 使用方法

1. `index.html`をブラウザで開く（または開発サーバーで実行）
   ```bash
   # 簡単なサーバーで実行する場合（Python）
   python -m http.server 8000
   
   # または Node.js の場合
   npx http-server
   ```

2. ブラウザで `http://localhost:8000` にアクセス

## 📋 操作方法

| 操作 | 説明 |
|------|------|
| テキスト入力 + Enter または 「追加」ボタン | 新しいタスクを追加 |
| チェックボックス | タスクの完了/未完了を切り替え |
| 「削除」ボタン | タスクを削除 |
| 「完了したタスクを削除」 | すべての完了済みタスクを一括削除 |

## 📁 ファイル構成

```
todo-app/

├── styles.css      # CSSファイル（スタイル）
```

## 🛠️ 技術スタック

- **HTML5** - マークアップ
- **CSS3** - スタイリング（グラデーション、アニメーション対応）
- **Vanilla JavaScript** - ロジック実装（フレームワーク不要）
- **LocalStorage** - ブラウザ内データ永続化

- シンプルで直感的なUI

- スムーズなアニメーション

## 💡 カスタマイズ

### 色を変更したい場合

`styles.css`内の以下の値を変更：
- `#667eea` - メインカラー（紫）
- `#764ba2` - アクセントカラー（濃い紫）
- `#ff6b6b` - 削除ボタンの色（赤）
- `#ffc107` - クリアボタンの色（黄色）

### フォントを変更したい場合

`styles.css`の最初の`body`セレクタ内の`font-family`を変更。

## 📝 ライセンス

このプロジェクトは自由に使用・修正できます。

## 🐛 トラブルシューティング

- **データが保持されない** → ブラウザのプライベートモードを使用していないか確認
- **スタイルが反映されない** → ブラウザキャッシュをクリアしてリロード（Ctrl+Shift+Delete）
- **Enterキーが反応しない** → 日本語入力がオンになっていないか確認
