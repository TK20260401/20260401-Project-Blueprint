# IT-Skills Learning OS (ISLOS)

### 「情報Ⅰ・Iパス・FE」を最短で攻略する、実践型アルゴリズム学習基盤

本日（2026/04/01）のキックオフに基づき、16年間の教育知見をシステム化。単なる暗記ではなく、シミュレーションを通じて「ITリテラシー」と「計算機科学」の核心を体得するプラットフォームの開発を開始します。

## 1\. プロジェクト概要 (Project Description)

共通テスト「情報Ⅰ」、ITパスポート、基本情報技術者試験のシラバスを完全に網羅。麻雀等の身近なゲームロジックを「生きた教材」として活用し、データの表現からネットワーク、セキュリティ、プログラミングまでを横断的に学習できるシステムを目指します。

## 2\. 目的 (Objective)

  - **資格・試験の完全攻略**:
      - **情報Ⅰ**: 共通テスト全4問（情報社会・デザイン・プログラミング・シミュレーション）への対応。
      - **ITパスポート**: ストラテジ・マネジメント・テクノロジの3分野を体系的に理解。
      - **基本情報 (FE)**: 科目B（アルゴリズム・擬似言語）の壁を視覚化で突破。
  - **自走自律学習 (SDL)**: AI伴走により「なぜその答えになるのか」のプロセスを重視した学習。
  - **教育DXの推進**: 学習ログを解析し、個々の理解度に応じた「個別最適な学び」を提供。

## 3\. 主要な機能と網羅される学習概念 (Core Features)

| 機能・機械的役割 | 詳細定義 (Extended Description) | ターゲット試験 / 概念 |
| :--- | :--- | :--- |
| **Data Structure Visualizer** | 牌や顧客データ等の「配列」「リスト」「スタック/キュー」の構造を動的に可視化。 | **Iパス/情報Ⅰ**: データの表現<br>**FE**: データ構造 |
| **Algorithm Debugger** | 探索（リサーチ）や整列（ソート）のコードが裏側でどう動いているかを1ステップずつ表示。 | **情報Ⅰ**: プログラミング<br>**FE**: アルゴリズム（科目B） |
| **Logic Logic Analyzer** | 複雑なルール（役判定、点数計算等）をIF文やCASE文の論理回路として図解。 | **Iパス**: テクノロジ系全般<br>**FE**: 論理演算 |
| **Security & Network Sim** | データの送受信プロセスや暗号化の仕組みをシミュレート。 | **情報Ⅰ**: 情報通信ネットワーク<br>**Iパス**: セキュリティ |
| **Statistical Analytics Engine** | 期待値計算や分散、相関分析をリアルタイム実行し、結果をグラフ出力。 | **情報Ⅰ**: データの活用（第4問）<br>**Iパス**: 表計算・統計 |
| **AI Personal Tutor** | 弱点データを蓄積し、次に解くべき問題をレコメンド。 | **教育DX**: 個別最適な学習 |

## 4\. 技術スタック (Tech Stack)

  - **Frontend**: `Next.js` / `Tailwind CSS` (アクセシビリティ・視覚化重視)
  - **Backend/DB**: `Supabase` (学習履歴の永続化、リアルタイム同期)
  - **AI Engine**: `Claude 3.5 Sonnet` / `Cursor` (教育的プロンプトの実行)
  - **Infrastructure**: `Vercel` (迅速なプロトタイプ配信)

## 5\. 開発ロードマップ (Roadmap)

1.  **Phase 1 (2026/04/01)**:
      - 「情報Ⅰ・Iパス」の重要語句と本システムの機能マッピング。
      - 最初のMVP（アルゴリズム視覚化モジュール）のデプロイ。
2.  **Phase 2**:
      - 過去問シミュレーション機能の実装。
      - Pythonを用いたデータ分析演習機能の追加。
3.  **Phase 3**:
      - チーム体制の最適化。教育現場での実証実験とフィードバック収集。

## 開発者メモ (For Developers)

  - **Modular Design**: 麻雀以外の題材（在庫管理、パズル等）にも容易に差し替え可能な「汎用アルゴリズムエンジン」として開発。
  - **Invisible Data**: ユーザーには「遊び」に見えるが、裏側では「試験の点数に直結する思考ログ」を確実に捕捉する。
