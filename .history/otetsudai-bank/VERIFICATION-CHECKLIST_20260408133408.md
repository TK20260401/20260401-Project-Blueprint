## 基本動作

### 1. ログイン
- [ ] 家族選択 → メンバー選択 → PIN入力で正常にログインできる
- [ ] 間違ったPINでログインが拒否される
- [ ] 親アカウント・子供アカウントそれぞれでログインできる

### 2. 親ダッシュボード（/parent）
- [ ] 承認待ち一覧が表示される
- [ ] 子供ごとのウォレット残高が表示される
- [ ] アカウント削除ボタンが存在する

### 3. 親タスク管理（/parent/tasks）
- [ ] タスクの新規作成ができる
- [ ] タスクの編集ができる
- [ ] タスクの削除ができる
- [ ] 繰り返し設定（毎日/毎週/一回のみ）が機能する
- [ ] 子供への割り当てができる

### 4. 子供ダッシュボード（/child/[id]）
- [ ] 🐷貯金箱の残高が正しく表示される
- [ ] タスク一覧が表示される
- [ ] 履歴タブに取引履歴が表示される

### 5. タスク承認フロー
- [ ] 子供がタスク完了報告できる
- [ ] 親の承認待ちに表示される
- [ ] 親が承認するとウォレット残高に反映される
- [ ] 親が差し戻しできる
- [ ] 承認は1秒以内に反映される
