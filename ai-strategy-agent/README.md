# AI Strategy Agent — AI戦略提案書 自動生成エージェント

## 目的

コンサルタント不在でもAIと伴走し、業務棚卸し・コスト削減・人月削減を実現するAIエージェント。企業名を入力するだけで、業務プロセス分析・AI導入提案・ROI算出を自動生成する。

## 独自の伴走プロセス（5フェーズ）

| Phase | 名称 | 内容 |
| --- | --- | --- |
| 01 | AI AGENTによる事前調査 | IR情報・採用情報・口コミ・ニュース・企業HPから情報収集、課題分析、AI導入提案のアウトラインを事前生成 |
| 02 | ヒアリングと業務分解 | 業務の棚卸し、承認プロセス、独自ノウハウ、データ活用ポイントを分解、ROI算出、導入プラン策定 |
| 03 | PoC/BPO実証 | 現場で実際に機能し自律的に運用される状態まで泥臭く伴走 |
| 04 | 現場導入・ラストワンマイル調整 | 試験導入、ケースによってはBPO運用、実業務の中でラストワンマイルの調整 |
| 05 | 定着支援 | 継続的サポート、バージョンアップ、さらなる効率化・最適化 |

## 主要な機能一覧

| # | 機能 | 概要 |
| --- | --- | --- |
| 1 | ダッシュボード | 総提案書数・今月の作成数・対象業界数・平均生成時間の集計 |
| 2 | 新規提案書作成 | 企業名入力→AIが事前調査→提案書自動生成 |
| 3 | 初回ヒアリング | 業務棚卸し・課題整理・要件定義をAIがアシスト |
| 4 | 業務プロセス可視化 | フェーズ毎・プロジェクト毎の進捗表示 |
| 5 | コスト削減分析 | 人員コスト・人月の削減シミュレーション |
| 6 | 提案書履歴 | 過去の提案書一覧・検索・フィルター |
| 7 | 分析レポート | 業界別・フェーズ別の分析ダッシュボード |

## 技術スタック（ベータ版）

| レイヤー | 技術 |
| --- | --- |
| フロントエンド | Next.js (App Router) on Vercel |
| UI | Tailwind CSS |
| バックエンド/DB | Supabase (PostgreSQL + Auth + RLS) |
| AI | Claude API (提案書生成・業務分析) |
| デプロイ | Vercel |
| 認証 | Supabase Auth |
