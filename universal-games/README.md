# ユニバーサルゲーム（あそびひろば）

老若男女、知的障害や肢体不自由のある方も含め、誰でも楽しめるゲーム集です。

## 対象ユーザー

- 未就学児〜高齢者
- 特別支援学校・支援学級・通級の利用者
- 脳性麻痺など微細動作が困難な方
- デイサービス・在宅で余暇を過ごしたい方

## アクセシビリティ

- 最小タップ領域 80×80pt 以上
- 文字サイズ3段階切替（だい／とくだい／ごくだい）
- 色＋形で区別（色覚多様性対応）
- 効果音 ON/OFF
- 失敗概念なし（時間制限・減点・ゲームオーバーなし）
- 全てひらがな表記

## 収録ゲーム

| ゲーム | 状態 |
|--------|------|
| もぐらたたき | プレイ可能 |
| すうじトレイル | プレイ可能 |
| ことばあそび | プレイ可能 |

## 技術スタック

- Expo SDK 54 + Expo Router
- TypeScript
- react-native-svg（SVG描画）
- @react-native-async-storage/async-storage（記録保存）
- expo-av（効果音）

## 起動手順

```bash
cd universal-games
npm install
npx expo start
```

起動後:
- **iOS**: `i` キーでシミュレータ起動、または Expo Go アプリでQRスキャン
- **Android**: `a` キーでエミュレータ起動、または Expo Go アプリでQRスキャン
- **Web**: `w` キーでブラウザ起動

## もぐらたたき設定

- はやさ: ゆっくり / ふつう / はやい
- 同時出現数: 1 / 2 / 3
- 穴の数: 4 / 9 / 16

## ことばあそび（しりとり画像版）

- 絵＋ひらがなで3択しりとり
- 正解でも不正解でも先に進む（失敗概念なし）
- 10問で1セッション
- 33語の辞書（高齢者にも親しみやすい日常語）

## すうじトレイル

- 1→2→3…と順番にタップして道をつなげる
- 難易度3段階: やさしい(5個) / ふつう(10個) / むずかしい(数字+ひらがな交互20個)
- タイマーON/OFF切替、ヒント点滅(3秒無操作で)
- 間違えても減点なし
