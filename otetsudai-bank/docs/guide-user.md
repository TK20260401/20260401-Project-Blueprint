# おこづかいクエスト ユーザーガイド

おこづかいクエストは、お手伝い（クエスト）を通じて「稼ぐ・貯める・増やす・使う」を家族で体験できるマネー教育アプリです。

---

## はじめかた

### 1. アカウント登録（親）

1. トップページ右上の「ログイン」→「新規登録」をタップ
2. メールアドレスとパスワード（8文字以上）を入力
3. 確認メールが届くので、メール内のリンクをタップして認証完了
4. 家族名・親の名前を登録
5. 子どもの名前とPIN（4桁の暗証番号）を登録

### 2. ログイン

| ロール | 方法 |
|--------|------|
| 親 | メールアドレス＋パスワード |
| 子ども | 家族を選択 → 名前をタップ → PIN入力 |

## 子ども向けガイド

### クエスト（おてつだい）をやろう

1. ダッシュボードの「クエストリスト」からやりたいクエストを選ぶ
2. おてつだいが終わったら「クリア！」ボタンをタップ
3. おうちの人がみとめたら、コインがもらえる

### じぶんクエストをていあんしよう

「こんなおてつだいもできるよ！」と思ったら：

1. 「じぶんクエスト」ボタンをタップ
2. やりたいおてつだいの名前とほうしゅう（コイン）を入力
3. おうちの人がOKしたら、クエストに追加されるよ

### コインのつかいかた

もらったコインは自動で3つにわかれる：

| 名前 | つかいみち |
|------|-----------|
| つかう | 好きなものを買うためのお金 |
| ためる | 目標にむけて貯金するお金 |
| ふやす | 投資のれんしゅう用のお金 |

#### お買いものをしたいとき

1. 「つかう」→「おかいもの申請」をタップ
2. 買いたいものと金額を入力
3. おうちの人がみとめたら、お金をもらえるよ

### ちょきん目標をつくろう

1. 「ためる」エリアの「＋目標をきめる」をタップ
2. ほしいもの名前と目標金額を入力
3. コインが貯まると進捗バーが進むよ

### とうしのれんしゅう

1. 「ふやす」エリアから銘柄（会社の名前）を選ぶ
2. 「買う」ボタンで注文を出す
3. おうちの人がみとめたら、株をもてるよ
4. 値段が上がったり下がったりするのを観察しよう

### バッジをあつめよう

クエストをたくさんこなしたり、コインを貯めたりすると、バッジがもらえるよ。ダッシュボード上部でコレクションを確認できます。

### コインくん（AIアシスタント）

画面右下のチャットボタンをタップすると、コインくんに相談できます。クエストのコツや、お金の使い方を教えてくれるよ。

## 親（保護者）向けガイド

### ダッシュボード

ログイン後、以下が一覧で確認できます：

- 承認待ちのクエスト完了報告
- 子どもの支出申請
- 各子どものウォレット残高（つかう／ためる／ふやす）

### クエスト（おてつだい）管理

「クエスト管理」画面（メニュー → クエスト管理）で：

1. **追加**: タイトル・報酬額・繰り返し（1回/毎日/毎週）・アイコンを設定
2. **編集**: 既存クエストの内容や報酬を変更
3. **削除**: 不要なクエストを削除

**報酬額の目安**

| 難易度 | 例 | 推奨額 |
|--------|-----|--------|
| かんたん | 靴をそろえる、箸を並べる | 30〜50円 |
| ふつう | 食器を洗う、洗濯物をたたむ | 100〜200円 |
| むずかしい | お風呂掃除、料理の手伝い | 200〜500円 |

### 承認ワークフロー

1. 子どもが「クリア！」→ 親のダッシュボードに通知
2. 内容を確認して「承認」or「やりなおし」
3. 承認時にスタンプ（褒め言葉）を送れる
4. 承認すると、設定した分配比率でコインが子どものウォレットに入る

### 分配比率の設定

報酬コインの分配割合を設定できます。

- 推奨の初期値: つかう70% / ためる20% / ふやす10%
- 子どもの年齢や目的に合わせて調整してください

### 支出申請の承認

1. 子どもの申請内容（品目・金額）を確認
2. 「承認」→ 支払い方法を選択（PayPay / B/43 / LINE Pay / 現金 / その他）
3. 実際に送金・手渡し後、「支払い済み」をタップ

### じぶんクエスト（子どもの提案）の対応

子どもが提案したクエストは親ダッシュボードに表示されます。

- **承認**: そのまま正式クエストに追加
- **金額変更**: 報酬額を調整して承認
- **却下**: 理由を添えてやりなおし

### 投資注文の承認

子どもの株の売買注文は親の承認が必要です。教育的な会話のきっかけとして活用してください。

### AIアドバイザー

画面右下のチャットボタンから、報酬設定や年齢別の教育方針についてAIに相談できます。

## よくある質問

**Q. 報酬は実際のお金ですか？**
A. アプリ内のコイン（仮想通貨）です。実際のお金への換算・送金は親が手動で行います。

**Q. ページをリロードしたらログアウトされた**
A. セキュリティ仕様です。ログイン画面から再度ログインしてください。

**Q. 子どものPINを忘れた**
A. 親アカウントでログインし、子どもの設定からPINを再設定できます。

**Q. 投資で損をしたらどうなる？**
A. シミュレーションなので実際のお金は減りません。「値段は上がったり下がったりする」ことを体験する教育機能です。

**Q. 対応ブラウザは？**
A. Chrome、Safari、Edge の最新版を推奨。PWA対応のため、ホーム画面に追加してアプリのように使えます。
