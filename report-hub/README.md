# Report Hub — 日報・週報・月報 統合レポートシステム

## 目的

「日報を積み上げれば週報になり、週報をまとめれば月報になる」粒度設計で、ITエンジニア（PM/PL候補）と教員の視点を融合した構造化報告基盤を構築する。工数削減アクションと教育知見（暗黙知）を定量化・形式知化して蓄積し、AI による振り返り自動生成にも対応する。

## 主要な機能一覧

| # | 機能 | 概要 |
| --- | --- | --- |
| 1 | 日報作成 | 実施タスク・成果・課題/対策・明日の予定・所要工数・工数削減アクション・定性的気づきを入力 |
| 2 | 週報自動集約 | 日報データを週単位で自動集約、KPI達成状況サマリー生成 |
| 3 | 月報自動集約 | 週報データを月単位で自動集約、KGI/KPI振り返り・トレンド表示 |
| 4 | 工数削減トラッカー | 「何をしたら何時間削減できたか」を記録・累計表示 |
| 5 | 暗黙知ナレッジベース | 定性的な教育知見・気づきを蓄積、タグ検索可能 |
| 6 | AI振り返り生成 | 蓄積データから「今週/今月の振り返り」を自動生成（構造化データ活用） |
| 7 | ダッシュボード | 工数推移グラフ・KPI達成率・ナレッジ蓄積量の可視化 |
| 8 | チーム共有 | PM/PLとしてナレッジシェアが最大化されるフォーマットで出力 |

## 技術スタック（ベータ版）

| レイヤー | 技術 |
| --- | --- |
| フロントエンド | Next.js (App Router) on Vercel |
| UI | Tailwind CSS |
| バックエンド/DB | Supabase (PostgreSQL + Auth + RLS) |
| デプロイ | Vercel |
| 認証 | Supabase Auth |
