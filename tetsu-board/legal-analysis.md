## 0. この記事の位置づけと免責

本稿は **公開情報および一般法律知識に基づく整理** であり、 **法的助言ではない**。最終的な判断は弁理士・弁護士に相談すること。商標登録状況は時期により変動するため、必ず最新の J-PlatPat 等で検証すること。

## 1. 商標法

### 1-1. 主要な検討対象の商標(推定)

| 商標 | 推定権利者 | 区分(指定商品・役務) | 当該プロジェクトとの関係 |
|---|---|---|---|
| 桃太郎電鉄 | コナミデジタルエンタテインメント株式会社 | 第9類(電子ゲームソフト)、第28類(玩具・ゲーム機等)他 | 完全独自名称のため抵触なし |
| 桃鉄(略称) | コナミ(同上、または別途略称商標) | 同上 | 同上 |
| ボンビー | コナミ(キャラクター標章) | 第9類他 | 完全独自キャラ名のため抵触なし |
| キングボンビー | コナミ(同上) | 同上 | 同上 |
| ハリーボンビー | コナミ(同上) | 同上 | 同上 |

※ 実際の権利者・登録区分は **J-PlatPat** で要確認(下記 5-1 参照)。

### 1-2. 商標権侵害の成立要件(商標法第25条・第37条)

以下を **すべて満たす場合に侵害成立**:

1. **商標としての使用**(出所識別機能としての使用)
2. **同一または類似の商標**(類否判断: 称呼・観念・外観の3要素)
3. **同一または類似の指定商品・役務**(商品分類との比較)

### 1-3. 独自作品(本プロジェクトの場合)の注意点

- 独自名称(`tetsu-board`)を使用 → 直接的な商標侵害リスクなし
- ただし **マーケティング表記** で他社商標を商標的使用すると問題化:
  - ❌「桃鉄風」「桃鉄をベースに」「ボンビー的存在」
  - ⭕「鉄道テーマのボードゲーム」「教育目的のターン制ボードゲーム」
- 商品比較・批評の範囲を超える使用は避ける

### 1-4. 推奨対応(本プロジェクト実施済)

- マーケ・広告・ゲーム内テキスト・コード内コメント・ドキュメント全般から特定先行作品名を排除
- 中立的表現への置換(「鉄道テーマ」「先行作品」等)
- 自社プロジェクト名(`tetsu-board`)の **商標出願検討**(リリース前)

## 2. 著作権法

### 2-1. アイデア・表現二分論(idea/expression dichotomy)

著作権は **表現を保護し、アイデアを保護しない**(著作権法の根本原則)。

| 著作権で保護される(表現) | 保護されない(アイデア・機能) |
|---|---|
| キャラクターの絵柄・デザイン | ゲームシステム・ルール |
| 固有名称(キャラ名・地名・固有用語) | 抽象的なゲームメカニクス |
| ストーリー(具体的展開) | 抽象的なテーマ |
| プログラムコード(具体的記述) | アルゴリズム |
| マップ・画面の具体的レイアウト | 一般的なUI構造 |
| 音楽・効果音・BGM | 楽曲のジャンル |

### 2-2. 翻案権(著作権法第27条)の射程

翻案の成立要件(判例の蓄積による):

- 原著作物の **表現上の本質的特徴を直接感得可能** であること
- 単なる **アイデアの共通** では翻案にならない

### 2-3. 本プロジェクトの判定

- ゲームメカニクス(サイコロ移動・物件購入・独占ボーナス等)が共通 → **保護対象外**、問題なし
- 完全独自のキャラクター絵・固有名称・ストーリー → 翻案にあたらない
- **結論: 著作権侵害リスクなし**

## 3. 不正競争防止法

### 3-1. 該当条項の検討

| 条項 | 内容 | 本プロジェクトとの関連 |
|---|---|---|
| 第2条1項1号(周知表示混同惹起行為) | 周知の他人の商品等表示と同一・類似で混同を生じさせる行為 | 「桃鉄風」マーケ表記が混同惹起の可能性 → 削除済 |
| 第2条1項2号(著名表示冒用行為) | 著名な他人の商品等表示の冒用 | 同上 |
| 第2条1項3号(商品形態模倣行為) | 他人の商品形態を模倣 | 該当なし(独自ビジュアル) |

### 3-2. 本プロジェクトの判定

- マーケ表記から特定先行作品名を排除済
- 完全独自のビジュアル・キャラ → 形態模倣に該当しない
- **結論: 不正競争防止法上のリスクなし**(対応済)

## 4. 参考判例

| 判例 | 裁判所/年 | ポイント |
|---|---|---|
| パックマン事件 | 東京地判昭和57年(1982年) | ゲーム画像・プログラムは著作物として保護される。ただしゲームの仕組みそのものは保護対象外 |
| ナムコ事件(コナミ vs ナムコ) | 大阪地判平成10年(1998年) | 似たメカニクスのゲームでも、具体的表現が独自なら著作権侵害にならない |
| たぬきの大冒険事件 | 東京地判昭和62年(1987年) | キャラクター類似性は「表現の本質的特徴の同一性」で判断 |
| 三菱地所事件(建築家) | 最判平成13年(2001年) | 翻案=「直接感得性」の基準を確立 |
| **釣りゲータウン2事件** | **知財高判平成24年(2012年)8月8日** | **ゲーム画面のレイアウト・具体的表現の類似性が問題に。「アイデアではなく具体的表現の比較」が判断ポイント** |

### 特に参考: 釣りゲータウン2事件

- グリー(原告)が DeNA(被告)の類似ゲームを著作権侵害で提訴
- 一審は侵害認定、知財高裁は侵害否定
- **論点**: 共通するのは「魚を釣る」という基本的アイデア+ゲーム画面の構成
- **判旨**: 「表現の本質的特徴を直接感得することはできない」として侵害否定
- **教訓**: メカニクス共通+画面構成類似でも、具体的表現が独自なら著作権侵害成立は困難

## 5. 実務チェックリスト

### 5-1. J-PlatPat 検索手順(無料・自分でできる)

1. https://www.j-platpat.inpit.go.jp にアクセス
2. 上部メニューから **「商標」** を選択
3. 「商標検索」 → 「商標(検索用)」欄に検索したい商標(例: 「桃太郎電鉄」)を入力
4. 「検索」ボタン
5. 検索結果一覧から該当商標を選択
6. 詳細表示で以下を確認:
   - 権利者(出願人/権利者)
   - 指定商品・役務(第何類、具体的範囲)
   - 登録日・存続期間
   - 防護標章登録の有無
7. 自社プロジェクトの商品/役務との重複を確認

### 5-2. MVP リリース前のチェックリスト

- [ ] J-PlatPat で関連商標(桃太郎電鉄・桃鉄・ボンビー・キングボンビー・ハリーボンビー)の登録状況確認
- [ ] 自社用途と重複する区分(指定商品・役務)の有無を確認
- [ ] マーケ・広告・ゲーム内テキスト・コード内コメント・ドキュメントから特定先行作品名を完全排除
- [ ] 自社プロジェクト名(`tetsu-board`)の商標出願検討
- [ ] **弁理士に商標調査依頼**(無料相談 or 1〜2万円程度の有料相談)
- [ ] 必要に応じて弁護士に著作権面の安全性確認(規模・展開先次第)

### 5-3. 弁理士・弁護士相談の目安

| 状況 | 推奨度 |
|---|---|
| 個人開発・無料配布のみ | 任意(自己責任) |
| 個人開発・課金あり(月数千円〜) | 任意〜推奨 |
| 法人事業 | **推奨** |
| 教育機関への直接導入(契約あり) | **必須** |
| 教育委員会・自治体納品 | **必須**(契約条件で求められる場合あり) |
