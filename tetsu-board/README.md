# tetsu-board

鉄道テーマのボードゲームジャンルにインスパイアされた **教育プロダクト**。第1弾は知的障害・発達系を含む特支/夜間定時制(C層)向け。教科横断(地理・経済・公民・情報・家庭・数学)の学習効果を、ボードゲームの楽しさで担保する。

## 設計原則(全フェーズに貫通)

1. **合理的配慮**: 障害者差別解消法準拠の配慮を全UI/UXに実装(漢字+ルビ・サウンド独立切替・ビジュアルサイズ調整・制限時間ユーザ選択・動的難易度・%表現切替・抽象→具体・行間ゼロ・ビジュアライズ強化等)
2. **9歳の壁対策**: 合理的配慮の中核となる対策(抽象→具体・行間ゼロ・ビジュアライズ強化、Phase 1〜)
3. **イニシアチブはユーザ**: 児童・生徒が主導権、教師は監督・枠設定(Phase 5 で明文化)
4. **余白を設ける**: 段階的拡張可能なアーキテクチャ・スケジュール(Phase 5 で明文化)
