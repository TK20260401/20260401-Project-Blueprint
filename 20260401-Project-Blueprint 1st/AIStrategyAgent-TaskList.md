#### 4-D. 計算ロジック・テストケース

- [ ] 4-D-1. 計算ロジックを関数として切り出し

```
計算式:
月間現状コスト = オペレーター人材コスト + 現行システム運用費 + (新人教育費用/12) + その他費用
月間AI導入後コスト = オペレーター人材コスト × (1 - 人員削減率) + AI月額費用 + (新人教育費用 × (1-教育削減率) /12) + その他費用
月間コスト削減額 = 月間現状コスト - 月間AI導入後コスト
年間効果額 = 月間コスト削減額 × 12
年間AI費用 = AI月額費用 × 12
ROI(%) = (年間効果額 - 年間AI費用) / (初期費用 + 年間AI費用) × 100
投資回収期間(月) = 初期費用 / 月間コスト削減額
```

- [ ] 4-D-2. テストケース3本

```
テストケース1: 標準ケース
入力: オペレーター2,000万, 人月80万, コール15,000件, 平均8分,
      システム300万, 教育500万, その他100万,
      初期6,000万, 月額200万, 自動化60%, 削減12人, 教育削減60%
期待: 月間削減額 > 0, ROI > 0, 回収期間 < 12ヶ月

テストケース2: 削減率0%（効果なし）
入力: 自動化0%, 削減0人, 教育削減0%
期待: 月間削減額 = 0, ROI < 0（投資のみ）

テストケース3: 高効率ケース
入力: 自動化90%, 削減20人, 教育削減80%
期待: ROI > 500%, 回収期間 < 3ヶ月
```
