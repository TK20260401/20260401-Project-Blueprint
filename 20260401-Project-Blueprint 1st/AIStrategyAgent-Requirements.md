# AI Strategy Agent — 要件定義書 v1.0

## 1. システム概要

### 1.1 システム名
AI Strategy Agent（AI戦略提案・収益計画シミュレーション・KGI/KPI管理 統合アプリ）

### 1.2 システムの目的
弊社の「AIエージェントシステム」をクライアントに導入してもらうために、費用対効果を簡単にシミュレーションできるWebアプリケーション。コンサルタントが不在でも、営業が現場でAIと伴走しながらROI・回収期間・年間効果額を即時可視化し、クライアントへの提案力を最大化する。

### 1.3 導入目的
- 営業がその場で投資対効果を数値で示し、クライアントの意思決定を加速する
- コンサルなしでもAIがアシスタントとして伴走し、提案品質を担保する
- KGI/KPIを一元管理し、導入後の効果を継続的に可視化する

## 2. ユーザー定義

### 2.1 ユーザー種別と権限

| ロール | 権限 | 想定人数 |
| --- | --- | --- |
| 管理者 | 全機能＋ユーザー管理（招待・削除・ロール変更）＋CSV一括登録 | 数名 |
| 営業 | シミュレーション作成・保存・編集・削除、KGI/KPI CRUD、提案書閲覧・PDF出力 | 数十名 |
| クライアント閲覧者 | シミュレーション結果・KGI/KPI閲覧のみ（編集不可） | ~200名 |

### 2.2 ユーザー登録方式
- 管理者がCSVインポートで一括登録
- 招待メールで各自がパスワード設定
- 自己登録は不可（管理者主導）

## 3. 機能要件

### 3.1 Phase 1（MVP）— 営業が提案時に使える状態

#### 3.1.1 認証・ユーザー管理
- Supabase Auth（メール/パスワード）
- ロール管理（管理者 / 営業 / クライアント閲覧者）
- 管理者によるユーザー招待・CSV一括登録
- ログイン/ログアウト

#### 3.1.2 収益計画シミュレーション
**入力項目（現状コスト）:**

| 項目 | 型 | 説明 |
| --- | --- | --- |
| オペレーター人材コスト | 万円/月 | 月間の人件費合計 |
| 人月のコスト | 万円/人月 | 1人月あたりの単価 |
| 月間コール件数 | 件/月 | 月間の問い合わせ・コール数 |
| 1コールの平均時間 | 分 | 1件あたりの平均対応時間 |
| 現行システム運用費 | 万円/月 | 既存システムのランニングコスト |
| オペレーター新人教育費用 | 万円/年 | 年間の教育・研修費用 |
| その他の費用 | 万円/月 | その他の関連コスト |

**入力項目（AI導入コスト）:**

| 項目 | 型 | 説明 |
| --- | --- | --- |
| 初期費用（一括） | 万円 | 導入・カスタマイズ費用を含む |
| 月額費用（ランニング） | 万円/月 | AI基盤の月額利用料 |

**入力項目（削減見込み）:**

| 項目 | 型 | 説明 |
| --- | --- | --- |
| AI自動化率 | % | コール対応のうちAIが自動化する割合 |
| 人員削減見込み | 人 | 削減可能な人数 |
| 教育コスト削減率 | % | 新人教育費用の削減割合 |

**出力（即時計算・即時可視化）:**

| 指標 | 計算方法 |
| --- | --- |
| 月間コスト削減額 | 現状コスト合計 - AI導入後コスト |
| 年間効果額 | 月間コスト削減額 × 12 |
| ROI（%） | (年間効果額 - 年間AI費用) / (初期費用 + 年間AI費用) × 100 |
| 投資回収期間 | 初期費用 / 月間コスト削減額（月数） |

**グラフ表示:**
- メイン: 折れ線グラフ — 累積コスト削減額 vs 累積投資額（交差点 = 回収完了）
- サブ: 棒グラフ — 月次ビフォーアフター比較（導入前 vs 導入後）
- 1画面に両方並べて表示

**操作:**
- スライダーまたは数値入力で値を変更 → 即座にグラフ・数値が更新
- シミュレーション結果の保存（名前をつけて保存）
- 保存済みシミュレーションの一覧・再編集・再計算
- 過去のシミュレーションを複製して条件変更

#### 3.1.3 KGI/KPI管理（目標設定）
- KGI（最終目標）の作成・編集・削除
  - 例: 年間コスト30%削減、ROI 200%達成
- KPI（中間指標）の作成・編集・削除
  - KGIに紐付く形で階層管理
  - 例: 月間AI対応率、平均応答時間短縮率、教育コスト削減率
- 目標値・期限の設定
- 一覧表示・フィルター

#### 3.1.4 ダッシュボード
- 統計カード: 総提案書数、今月の作成数、対象業界数、平均生成時間
- 最近のシミュレーション一覧
- KGI/KPI進捗サマリー

#### 3.1.5 提案書管理
- 企業情報入力（企業名・業界・従業員数・売上・課題・対象プロセス）
- 5フェーズ伴走プロセスの進捗管理
- 提案書一覧（履歴）・検索・フィルター

### 3.2 Phase 2（実績トラッキング）— 導入後

#### 3.2.1 KGI/KPI実績値入力
- 月次の実績値を入力
- 目標 vs 実績の進捗率を自動計算
- 時系列グラフ（目標線 vs 実績線）

#### 3.2.2 分析レポート
- 業界別・フェーズ別の分析
- コスト削減トレンド

### 3.3 Phase 3（拡張）

#### 3.3.1 PDF出力
- シミュレーション結果をPDF形式でエクスポート
- 商談時にそのまま印刷可能なレイアウト

#### 3.3.2 共有URL
- 閲覧専用リンクの発行
- クライアントがログインなしで結果を閲覧

## 4. 独自の伴走プロセス（5フェーズ）

```
01 AI AGENTによる事前調査
│  IR・採用・口コミ・ニュース・企業HPから情報収集
│  課題分析、AI導入提案のアウトラインを事前生成
│
02 ヒアリングと業務分解
│  業務の棚卸し、承認プロセス、独自ノウハウ分解
│  ROI算出、実現性のある導入プラン策定
│
03 PoC/BPO実証
│  現場で実際に機能し自律運用される状態まで伴走
│
04 現場導入・ラストワンマイル調整
│  試験導入、BPO運用、実業務での調整
│
05 定着支援
   継続サポート、バージョンアップ、最適化
```

## 5. 非機能要件

### 5.1 対応デバイス
- PC（メイン）
- タブレット（営業の外出先）
- スマートフォン（閲覧・確認用）
- レスポンシブデザイン必須

### 5.2 パフォーマンス
- シミュレーション計算: 入力変更から200ms以内に結果反映
- ページ読み込み: 3秒以内

### 5.3 セキュリティ
- Supabase RLS（Row Level Security）でデータアクセス制御
- ロールベースの権限管理
- HTTPS通信
