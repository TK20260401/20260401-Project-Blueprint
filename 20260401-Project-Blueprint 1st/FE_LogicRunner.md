**基本情報技術者試験（FE）**、特に配点の高い **「科目B（アルゴリズムとプログラミング、情報セキュリティ）」** に完全にフォーカスした自走自律型Webアプリの構成案を作成しました。

FEの科目Bは、従来の暗記型から「初見のアルゴリズムをその場で解読する力」が問われる形式に変わりました。ここを攻略するための **「動く疑似言語トレース環境」** を核に据えます。

---

## プロジェクト構成案：FE-LogicRunner（エフイー・ロジックランナー）

### 1. タイトル
**FE-LogicRunner ー 基本情報「科目B」アルゴリズム完全攻略シミュレーター**

### 2. 目的
基本情報技術者試験（FE）の科目Bで合否を分ける「疑似言語」の読解力を、実際にコードを動かし、変数の遷移を視覚化（トレース）するプロセスを通じて、自律的に習得する。

### 3. 主要な機能一覧

| 機能名 | 詳細・役割 | FE試験（科目B）への直結要素 |
| :--- | :--- | :--- |
| **FE疑似言語インタープリタ** | 試験独特の表記（`if`, `while`, `return`等）をそのまま入力して実行できるエディタ。 | 疑似言語の構文理解と実行順序の把握。 |
| **動的データ構造ビジュアライザ** | スタック、キュー、リスト、二分探索木などのデータの動きをアニメーション化。 | 「データ構造とアルゴリズム」分野の視覚的理解。 |
| **ステップ実行・トレース表** | 1行実行するごとに、全変数の値の変化を履歴として一覧表示。 | 最も時間のかかる「トレース作業」の自動化と答え合わせ。 |
| **セキュリティ・ロジック演習** | 公開鍵暗号、共通鍵暗号、デジタル署名の仕組みをシミュレート。 | 科目Bのもう一つの柱「情報セキュリティ」対策。 |
| **AIロジック・メンター** | 「このループが終了しない原因は？」など、論理的な問いかけで自走を支援。 | 未知のアルゴリズムを解読する思考力の育成。 |

---

### 4. 具体的な学習コンテンツ例（自走自律の仕掛け）

16年間の教育実践（個別最適化）をFEの難所にぶつけます。

* **再帰関数の可視化**:
    * 階乗計算やフィボナッチ数列などを例に、関数が自分自身を呼び出し、スタックに積まれていく様子を視覚化。
* **探索と整列の効率比較**:
    * 線形探索と二分探索、あるいは各種ソートアルゴリズムを同時に走らせ、比較回数や計算量（$O(n)$, $O(\log n)$等）の差をグラフで表示。
* **ビット演算とシフト操作**:
    * 論理積(AND)、論理和(OR)、排他的論理和(XOR)や左シフト・右シフトによる数値の変化をリアルタイムで確認。

---

### 5. 技術スタック（ベータ版：Vercel / Supabase）

* **Frontend**: **Next.js (Vercel)**
    * TypeScriptによる厳密な型管理。
    * Framer Motion等を用いた、アルゴリズム挙動のアニメーション実装。
* **Backend/Database**: **Supabase**
    * `fe_problems`：シラバスに沿った疑似言語のサンプル問題集。
    * `user_snippets`：ユーザーが自分で作成・保存したアルゴリズムの断片。
* **Development Support**: **Cursor / Claude code**
    * 疑似言語のパース（解析）ロジックの高速実装。
