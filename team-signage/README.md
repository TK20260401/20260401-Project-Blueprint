# Team Signage — チーム状況リアルタイム共有デジタルサイネージ

## 概要

少人数チーム（10〜20名）の状況を「デジタルサイネージ」でリアルタイムに共有するWebアプリケーション。一目で「誰が・どこで・何をしているか」がわかる視認性を重視。

## 機能一覧

| 機能 | 内容 |
| --- | --- |
| 在席管理（ステータスボード） | 氏名・現在地・勤務形態・一言メモ。色分けバッジで在席/外出/会議中/休み/接客を即判別。10〜20名のカード一覧表示 |
| 営業カレンダー | 週間カレンダー形式+直近6件の予定リスト。今日・明日の予定を強調表示。訪問先・予定内容を一目で把握 |
| リソース管理（キャパシティ可視化） | 案件/現場ごとの担当者・稼働率をバーチャートで表示。リソース不足や空きが一目でわかるUI |

## ステータス配色ルール

| ステータス | 色 | 用途 |
| --- | --- | --- |
| 在席 | 青（#3b82f6） | オフィスに在席中 |
| 外出 | 赤（#ef4444） | 外出・訪問中 |
| 会議中 | 紺（#1e3a5f） | 会議室で打ち合わせ中 |
| 休み | グレー枠 | 休暇・欠勤 |
| 接客 | 紫（#7c3aed） | 来客対応中 |
| 車 | 暗赤（#991b1b） | 社用車で移動中 |

## アーキテクチャ

```
デジタルサイネージ（ブラウザ）
    ↑ fetch() / 30〜60秒ポーリング
    |
GAS Web App（doGet → JSON）
    ↑
Google Spreadsheet（3シート）
    ├── 在席管理シート
    ├── 営業カレンダーシート
    └── リソース管理シート
```

## データ項目

### 在席管理シート

| カラム | 型 | 説明 |
| --- | --- | --- |
| name | TEXT | 氏名 |
| status | TEXT | 在席/外出/会議中/休み/接客 |
| sub_status | TEXT | サブステータス（車など） |
| location | TEXT | 現在地（拠点/フロア） |
| work_type | TEXT | 勤務形態（出社/在宅/外出/出張） |
| memo | TEXT | 一言メモ |
| updated_at | DATETIME | 最終更新日時 |

### 営業カレンダーシート

| カラム | 型 | 説明 |
| --- | --- | --- |
| date | DATE | 予定日 |
| name | TEXT | 担当者名 |
| content | TEXT | 予定内容 |
| type | TEXT | 種別（訪問/会議/出張/その他） |
| location | TEXT | 場所・訪問先 |

### リソース管理シート

| カラム | 型 | 説明 |
| --- | --- | --- |
| project_name | TEXT | 案件名/現場名 |
| assigned_members | TEXT | 担当者（カンマ区切り） |
| utilization_rate | NUMBER | 稼働率（0-100%） |
| status | TEXT | 進行中/完了/待機 |
| deadline | DATE | 期限 |

## 運用設計 — 入力の簡略化

| 工夫 | 内容 |
| --- | --- |
| スプレッドシート直接編集 | ログイン不要。スプレッドシートを開いてセルを書き換えるだけ |
| ドロップダウン入力 | ステータス列はデータ入力規則でプルダウン選択（在席/外出/会議中/休み/接客） |
| スマホからの更新 | Google Sheetsアプリで外出先からもステータス変更可能 |
| 自動リフレッシュ | サイネージ側は自動ポーリング。手動操作一切不要 |
| テンプレート行 | メンバー行は事前に用意。新規追加は行コピーのみ |
